# Network

Social network website

[YouTube](https://www.youtube.com/watch?v=xg2dGn0i5Ng) |
[Project Specification](https://cs50.harvard.edu/web/2020/projects/4/Network/)

Implements:
- Login system
- All posts page
- Following posts page
- Profile page
- Create post
- Edit post
- Follow/Unfollow
- Like/Unlike
- Pagination

Uses:
- Django
- HTML
- CSS
- Python
- JavaScript (Vanilla)
